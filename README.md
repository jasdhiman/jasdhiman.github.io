# Jaskaran Dhiman academic website

This folder contains a GitHub Pages-ready academic website for `jasdhiman.github.io`.

## Files

- `index.html` — main website content
- `style.css` — website design and responsive layout
- `script.js` — smooth scrolling, mobile menu, reveal animations, active navigation
- `assets/` — place your CV PDF and optional profile photo here

## How to upload to GitHub

1. Go to your repository: `jasdhiman.github.io`
2. Replace the existing `index.html`, `style.css`, and `script.js` with the files in this folder.
3. Commit the changes.
4. Wait a few minutes for GitHub Pages to update.

## CV button

The homepage includes a "Download CV" button pointing to:

`assets/Jaskaran_Dhiman_CV.pdf`

To make this button work, upload your final CV PDF to the `assets` folder using exactly this filename:

`Jaskaran_Dhiman_CV.pdf`

If you do not want a CV button yet, remove this line from `index.html`:

```html
<a class="btn btn-light" href="assets/Jaskaran_Dhiman_CV.pdf" target="_blank" rel="noopener">Download CV</a>
```
